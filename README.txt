Стек:
  - Vue js;
  - HTML;
  - JS;
  - Css;

  - Python 3: 
      - flask; 
      - sqlite3;
      - json.

Для запуска вам необходимо клонировать репозиторий, установить необходимые пакеты и языки программирования.
После клонирования необходимо ввести команду npm i из папки app;
Затем в деррикторие проекта откройте терминал и запустине из папки backend - python3 app.py и из папки app - npm run dev
